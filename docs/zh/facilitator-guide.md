# 讲师指南

> 研讨会讲师内部指南。请勿与参与者分享。

---
## 概述

这是一个包含 **4 个模块、约 3.5 小时的实战工作坊**，专为 agy-cli (Antigravity CLI) 设计。它面向开发者群体：包括正在评估或采用 agy-cli 的工程师、技术主管和解决方案架构师。

## 逐模块交付说明

### 模块 1 — SDLC 生产力提升 (50 分钟)

**核心信息：** agy 消除了在陌生代码库中导航的心智负担。它不是自动补全 — 它是你可以询问任何问题的高级工程师。

- **先演示，后练习。** 在要求参与者尝试他们自己的代码库之前，先在你自己的代码库上进行 1.1 节（代码理解）的现场演示。
- **常见阻力：** 参与者试图编写完美的提示词。鼓励使用自然语言。“告诉我 auth 是如何工作的”比“请解释这个代码库的身份验证架构”更好。
- **AGENTS.md 时刻：** 1.5 节是一个高价值的演示。在屏幕上实时创建一个 AGENTS.md，并展示下一个会话如何立即变得更智能。

### 模块 2 — 插件生态系统 (45 分钟)

**核心信息：** 只需一条 `agy plugin import gemini` 命令，即可将你在 Gemini CLI 中构建的所有内容迁移过来。你的扩展投资将得以保留。

- **现场演示 `agy plugin import gemini`** — 让观众看到真实的输出。这在视觉上非常有说服力。
- **验证练习：** 插件验证部分最好使用 `samples/plugins/workshop-helpers/` 中的示例插件进行。
- **市场占位符：** 如果被问及 `plugin install`，请说明该功能即将推出，并引导他们使用 `plugin import` 作为当前路径。

### 模块 3 — DevOps 编排与自动化 (40 分钟)

**核心信息：** `--print` 是安全出口。一旦你可以通过管道传输 agy 的输出，你就可以自动化任何事情。

- **最佳演示：** `git diff --cached | agy -p "Review for bugs."` — 每个开发者都能立刻看到它的价值。
- **CI/CD 部分：** 不要现场编写完整的工作流。展示 GitHub Actions 的 YAML 文件，并解释 `--dangerously-skip-permissions` 模式。
- **跳过沙盒深度剖析**，除非观众关注安全/合规性。

### 模块 4 — 多代理与高级模式 (45 分钟)

**核心信息：** 子代理 + /btw 是质的飞跃。这就是 agy 成为编排器而不仅仅是聊天机器人的地方。

- **子代理演示是令人惊叹的时刻。** 现场生成两个代理，展示它们同时运行。
- **/btw 演示：** 开始一个较长的任务（重构一个文件），然后在任务中途使用 /btw。向参与者展示在合并注入的注释时，光标会继续移动。
- **调度：** 从概念上描述该模式，不要进行现场演示（延迟会使工作坊中的演示变得尴尬）。

## 常见参与者问题

| 问题 | 解答 |
|---|---|
| "agy 使用什么模型？" | 使用 `/model` 查看和切换。请参阅 [模型文档](https://www.antigravity.google/docs/models)。 |
| "这与 Gemini CLI 有什么不同？" | agy 桥接了 Gemini CLI 和 Claude 的插件，具有原生的子代理编排，以及 `/btw` 任务中途引导。这是不同的产品。 |
| "我可以使用自己的 API 密钥吗？" | agy 使用基于浏览器的 Google 登录。企业级用户连接 GCP 项目。请参阅 [企业级文档](https://www.antigravity.google/docs/enterprise)。 |
| "代码会发送给 Google 吗？" | 有关数据处理的详细信息，请参阅 [常见问题解答](https://www.antigravity.google/docs/faq)。 |
| "关于钩子呢？" | agy-cli 通过 `hooks.json` 支持钩子。请参阅 [钩子文档](https://www.antigravity.google/docs/hooks)。 |
| "对话日志存储在哪里？" | `~/.gemini/antigravity-cli/conversations/` |

## 研讨会期间的故障排除

| 症状 | 解决方法 |
|---|---|
| `agy: command not found` | 检查 PATH 环境变量。运行 `which agy` 或 `which agy-cli`。 |
| 身份验证错误 / 401 | 会话凭据可能已过期。重新分发身份验证信息。 |
| `agy plugin list` 错误 | 检查 `~/.gemini/antigravity-cli/` 是否存在 |
| 响应缓慢 | 检查网络。空闲后的首次运行可能会因为工作区索引而变慢。 |
| 子代理未生成 | 确认参与者处于交互模式（而不是 `--print`） |
